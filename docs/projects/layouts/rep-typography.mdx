### **Spatial Architecture and Margin Configuration**

  | Margin Edge | Specification |
  | :--- | :--: |
  | Left (Binding) | `0.75` inches |
  | Right | `0.75` inches |
  | Top | `1.00` inch |
  | Bottom | `1.00` inch |

---

### **Typographic Hierarchy and Font Scaling**

| Hierarchy Level | Point Size | Font Weight & Case | Alignment | Vertical Spacing (Before / After)  |
|-----------------|------------|---------------------|-----------|-----------------------------------|
| H1 (Chapter)    | 20pt       | Bold, ALL CAPS      | Centered  | 0pt(Use page break) / 12pt        |
| H2 (Section)    | 18pt       | Bold, Title Case    | Left      | 18pt / 12pt                       |
| H3 (Sub-section)| 16pt       | Bold, Sentence Case | Left      | 12pt / 6pt                        |
| B1 (Bullet 1)   | 14pt       | Bold, Sentence Case | Left      | 12pt / 6pt                        |
| B2 (Bullet 2)   | 12pt       | Regular, Sentence Case | Justified | 0pt / 6pt (1.5 Line Spacing)   |
| H4 (Minor)      | 14pt       | Bold, Underlined    | Left      | 12pt / 6pt                        |
| Normal Text     | 12pt       | Regular, Sentence Case | Justified | 0pt / 6pt (1.5 Line Spacing)   |

---

### **Paragraph Architecture and Inter-Paragraph Spacing**

| Heading Level	| TOC Level	| Left Indentation | Text Formatting | Inclusion Status|
| --- | --- | --- | --- | --- |
| Chapter Title (H1)	| TOC 1	| 0.00 inches	| 12pt, Bold	| Mandatory |
| Main Heading (H2)	| TOC 2	| 0.50 inches	| 12pt, Regular	| Mandatory |
| Sub-heading (H3)	| TOC 3	| 1.00 inches	| 12pt, Regular	| Mandatory |
| Minor Heading (H4)	| TOC 4	| N/A	| N/A	| Excluded |

---

### **Structural Anatomy, Pagination, and Front Matter**

| Document Section	| Component Title	| Pagination Format	| Required |
| --- | --- | --- | --- |
| Front Matter | Title Page	| Implicit (i) - Not printed | True |
| Front Matter | Certificate & Declaration	| Roman (ii, iii)	| True |
| Front Matter | Acknowledgement	| Roman (iv)	| True |
| Front Matter | Abstract	| Roman (v)	| True |
| Front Matter | Table of Contents	| Roman (vi, vii)	| True |
| Front Matter | List of Figures & Tables	| Roman (viii, ix)	| False |
| Main Body	| Chapters 1 through N	| Arabic (1, 2, 3...)	| True |
| Back Matter	| Appendices & Source Code	| Arabic (Continued)	| False |

```txt
=============================================================================
         OPTIMIZED PROJECT REPORT FORMATTING SPECIFICATIONS
=============================================================================
1. PAGE DIMENSIONS & MARGINS (A4 Size, physically configured for Spiral Binding)
-----------------------------------------------------------------------------
Left Margin (Binding/Gutter) : 1.75 inches (or 1.0" margin + 0.5" gutter)
Right Margin                 : 1.75 inches
Top Margin                   : 1.00 inches
Bottom Margin                : 1.00 inches

2. TYPOGRAPHIC HIERARCHY (Typeface: Times New Roman strictly enforced)
-----------------------------------------------------------------------------
H1 (Chapter Titles)          : 20pt, BOLD, ALL CAPS, Centered Alignment
                               Spacing: 30pt Before, 30pt After. 
                               Constraint: Must force Page Break before.
H2 (Main Headings)           : 18pt, BOLD, Title Case, Left Aligned
                               Spacing: 18pt Before, 12pt After.
H3 (Sub-Headings)            : 16pt, BOLD, Sentence/Title Case, Left Aligned
                               Spacing: 12pt Before, 6pt After.
H4 (Paragraph Headings)      : 14pt, BOLD (or Underlined), Left Aligned
                               Spacing: 12pt Before, 6pt After.
Normal (Body Text)           : 12pt, Regular, Justified Alignment
                               Line Spacing: 1.5 lines
                               Paragraph Spacing: 0pt Before, 6pt or 12pt After
                               Paragraph Indent: 0.5" First Line OR Block (0")

3. TABLE OF CONTENTS (TOC) NAVIGATIONAL STYLING
-----------------------------------------------------------------------------
Font Specification           : 12pt, Times New Roman
Line Spacing                 : 1.5 lines
Level 1 Indent (Chapters)    : 0.00" (Bold Text, 12pt space before)
Level 2 Indent (H2s)         : 0.50" (Regular Text)
Level 3 Indent (H3s)         : 1.00" (Regular Text)
Dot Leaders                  : Tab Stop mathematically placed at exact right 
                               margin constraint (e.g., 5.77" for A4), 
                               Right-Aligned, Leader Style "..."

4. DATA PRESENTATION: TABLE AND FIGURE FORMATTING (IEEE Standard)
-----------------------------------------------------------------------------
Table Captions               : 10pt-11pt, Centered, positioned ABOVE the table.
Figure Captions              : 10pt-11pt, Centered, positioned BELOW the figure.
Table Borders                : NO vertical lines permitted. Horizontal lines 
                               utilized only above/below headers and bottom edge.
Table Data Alignment         : Left align text strings. Center numerical data. 
                               Strictly align decimals vertically.
Code Snippets (Appendices)   : 10pt Courier New or Consolas, Single Spaced (1.0).

5. PAGINATION, DOCUMENT SECTIONS, & REFERENCING
-----------------------------------------------------------------------------
Front Matter (TOC, Abstract) : Lowercase Roman numerals (i, ii, iii...), Bottom Center.
Main Body (Ch. 1 onwards)    : Arabic numerals (1, 2, 3...), Bottom Center.
References Citation Style    : IEEE Style, bracketed  numbering inline.
=============================================================================
```
