### **Standard Format for Experiments**

- **Aim:**  
  *[A one-line objective of the practical session]*
- **Theory / Prerequisite:**  
  *[A very brief description of the concepts or utilities being used]*
- **Commands Used:**  
  *[List the primary commands with their basic syntax, e.g., `mkdir <directory_name>`]*
- **Execution / Procedure:**  
  *[The actual terminal inputs. Leave space here to stick printed screenshots of your terminal outputs, or write the terminal output manually if required by your college.]*
- **Conclusion:**  
  *[A brief concluding sentence stating that the commands/scripts were successfully executed and understood.]*

### **List of Experiments**

**Unit I: Introduction & Terminal Commands**

- **Experiment 1:** Familiarization with basic Linux terminal commands for system navigation and directory management (`ls`, `pwd`, `cd`, `mkdir`, `rmdir`).
- **Experiment 2:** File manipulation and modifying file permissions (`touch`, `cp`, `mv`, `rm`, `chmod`, `chown`, `chgrp`).

**Unit II: File Operations & Text Processing**

- **Experiment 3:** Working with text editors and basic file operations (`cat`, `nano`, `vi`).
- **Experiment 4:** Text processing, filtering, and input/output redirection (`grep`, `awk`, `sed`, `cut`, `sort`, `uniq`, `>`, `>>`, `|`).

**Unit III: Process Management**

- **Experiment 5:** Monitoring, managing, and terminating system processes (`ps`, `top`, `htop`, `kill`, `nice`, `fg`, `bg`).
- **Experiment 6:** Scheduling automated tasks and managing user/group accounts (`cron`, `at`, `useradd`, `usermod`, `passwd`, `groupadd`).

**Unit IV: Package Management & Networking**

- **Experiment 7:** Installing, updating, and removing software packages (using apt/yum/dnf).
- **Experiment 8:** Network configuration, troubleshooting, and remote access (`ifconfig`, `ping`, `netstat`, `wget`, `ssh`, `scp`).

**Unit V: Disk Management & Shell Scripting**

- **Experiment 9:** Disk usage monitoring and managing partitions/file systems (`df`, `du`, `fdisk`, `mkfs`, `mount`).
- **Experiment 10:** Writing basic bash shell scripts (using variables and taking user input).
- **Experiment 11:** Writing advanced shell scripts implementing conditional statements, loops, and functions.
