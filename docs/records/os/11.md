### **Aim**
> To write basic bash shell scripts using variables and taking user input

### **Theory**
- **Shell Script Basics:**A script is a text file containing a sequence of commands. It begins with a "shebang" (`#!/bin/bash`) to define the interpreter.

- **`echo`**
  - This command prints the given input string to standard output.
  - *Syntax:* `echo [options..] [string]`

- **Variables & Input:** Variables temporarily store data in memory. User input is captured from the terminal using the read command. Variables are referenced using the $ symbol (e.g., `$var`).

- **Arithmetic Operators:** Used to perform mathematical calculations. Common operators include `+` (Addition), `-` (Subtraction), `*` (Multiplication), `/` (Division), and `%` (Modulus).

### **Commands**
```bash

$ cat > basic_script.sh
#!/bin/bash
# A basic script to take user input and perform arithmetic

echo "Enter your name: "
read username

echo "Welcome to the OS Lab, $username!"
echo "Let's do some basic math."

echo "Enter the first number: "
read num1
echo "Enter the second number: "
read num2

sum=$(($num1 + $num2))
echo "The sum of $num1 and $num2 is: $sum"
(Press Ctrl+D to save)

$ chmod +x basic_script.sh
$ ./basic_script.sh
Enter your name: 
Student
Welcome to the OS Lab, Student!
Let's do some basic math.
Enter the first number: 
10
Enter the second number: 
20
The sum of 10 and 20 is: 30

```


### **Conclusion**
> A basic bash shell script was successfully created and executed. The script demonstrated how to prompt for and capture user input (`read`), store data in variables, and perform basic arithmetic operations natively within the shell environment
