### **Aim**
> To study and execute commands for network configuration, troubleshooting, and remote access


### **Theory**
- **`ifconfig` / `ip`**
  - Displays or configures network interface parameters (like IP addresses and MAC addresses). ip is the modern networking command that replaces ifconfig.
  - *Syntax:* `ifconfig` or `ip addr show`.

- **`ping`**
  - Tests network connectivity between the local system and a remote host by sending ICMP ECHO_REQUEST packets.
  - *Syntax:* `ping -c [count] [hostname_or_IP]`.

- **`netstat` / `ss`**
  - Displays active network connections, routing tables, and listening ports.
  - *Syntax:* `netstat -tuln` (shows listening TCP/UDP ports)

- **`wget` / `curl`**
  - Command-line network downloaders used to fetch files from the internet using HTTP, HTTPS, or FTP protocols.
  - *Syntax:* `wget [URL]` or `curl [URL]`

- **`ssh` (Secure Shell)**
  - A cryptographic network protocol used for secure remote login and command execution on another Linux machine.
  - *Syntax:* `ssh [username]@[remote_ip_address]`

- **`scp` (Secure Copy)**
  - Securely transfers files between a local host and a remote host over the SSH protocol.
  - *Syntax:* `scp [source_file] [username]@[remote_ip]:[destination_path]`
