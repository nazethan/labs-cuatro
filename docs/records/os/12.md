### **Aim**
> To write advanced shell scripts implementing conditional statements, loops, and functions


### **Theory**
- **Conditional Statements (`if-else`):** Used to execute code blocks based on whether a condition evaluates to true or false.
  - *Syntax:* `if [ condition ]; then ... else ... fi`. It heavily relies on relational operators like `-eq` (Equality) and `-gt` (Greater than).

- **Loops (`while`, `for`):** The while loop repeatedly executes a block of commands as long as the given condition remains true.

- **Functions:** Blocks of reusable code defined once and called multiple times within the script.
  - *Syntax:* `function_name() { ... commands ... }`


### **Commands**
```bash

$ cat > advanced_script.sh
#!/bin/bash

# Function definition to check if a number is a palindrome
check_palindrome() {
    num=$1
    temp=$num
    s=0
    rev=""

    # While loop to reverse the number
    while [ $num -gt 0 ]
    do
        s=$(($num % 10))
        num=$(($num / 10))
        rev=$(echo ${rev}${s})
    done

    # Conditional statement to check equality
    if [ $temp -eq $rev ]; then
        echo "Number $temp is a palindrome"
    else
        echo "Number $temp is NOT a palindrome"
    fi
}

# Main script execution taking user input
echo "Enter a number to check:"
read input_num

# Calling the function and passing the input as an argument
check_palindrome $input_num
(Press Ctrl+D to save)

$ chmod +x advanced_script.sh

$ ./advanced_script.sh
Enter a number to check:
545
Number 545 is a palindrome

$ ./advanced_script.sh
Enter a number to check:
123
Number 123 is NOT a palindrome

```


### **Conclusion**
> An advanced shell script was successfully written and executed to check if a given number is a palindrome. The script effectively demonstrated the practical implementation of logical looping (`while`) , strict decision-making conditionals (`if`, `-eq`, `-gt`), and modular code organization using bash functions.
